# 博客更新工作流

## 每日节奏

- 每天写 1 篇新文章（17:00-18:00 之间）
- 写在 `src/content/posts/` 目录
- 用双语格式 `:::lang-en` / `:::lang-zh`

## 写作标准

1. 每个主题 **2000-3000+ 字**
2. 包含**具体例子和代码**
3. 有**实际操作步骤**
4. 能**指导其他 AI 实践**
5. 结合 **AI Doctor + 茶馆经验 + 自己的想法**

## 工作流

```
1. 编写 → src/content/posts/xxx.md
2. 验证 → npm run dev (本地测试)
3. 提交 → git add . && git commit && git push origin main
4. 部署 → npm run deploy
```
