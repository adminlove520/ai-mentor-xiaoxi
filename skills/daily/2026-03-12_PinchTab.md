## 1. 介绍

**PinchTab** 是一个轻量级的浏览器控制工具，专为 AI agents 设计！

- 6.8k stars 超受欢迎
- Token 高效：800 tokens/page（比截图便宜 5-13 倍）
- 稳定性好：使用稳定的 accessibility refs
- 支持多实例：可同时运行多个浏览器实例

---

## 2. 配置

### 安装 PinchTab

```bash
# macOS / Linux
curl -fsSL https://pinchtab.com/install.sh | bash

# 或 npm
npm install -g pinchtab
```

### 启动服务

```bash
pinchtab &
```

服务启动后访问 http://localhost:9867

---

## 3. 使用

### 基础命令

```bash
# 打开网页
pinchtab nav https://example.com

# 获取页面结构（推荐使用 -i -c 参数，最省 tokens）
pinchtab snap -i -c

# 点击元素（通过 ref）
pinchtab click e5

# 输入文字
pinchtab fill e3 "搜索内容"

# 提交表单
pinchtab press e7 Enter

# 截图
pinchtab screenshot
```

### 完整示例：搜索任务

```bash
# 1. 打开 Google
pinchtab nav https://google.com

# 2. 获取页面
pinchtab snap -i -c

# 3. 在搜索框输入（假设 ref 是 e5）
pinchtab fill e5 "什么是 AI"

# 4. 按回车
pinchtab press e5 Enter

# 5. 获取结果
pinchtab snap -i -c
```

---

## 4. 推荐理由

1. **省 Token**：800 tokens/page，比传统截图便宜 5-13 倍
2. **操作稳定**：使用 accessibility refs，不用担心元素定位问题
3. **轻量快速**：12MB 二进制文件，启动快
4. **本地安全**：不联系外部服务，数据不外泄
