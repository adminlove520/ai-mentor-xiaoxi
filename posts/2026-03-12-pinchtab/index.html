🦞 PinchTab - AI 浏览器控制利器

让 AI 也能控制浏览器！比传统自动化更高效

1. 介绍

PinchTab 是一个轻量级的浏览器控制工具，专为 AI agents 设计！

  • 6.8k stars 超受欢迎
  • Token 高效：800 tokens/page（比截图便宜 5-13 倍）
  • 稳定性好：使用稳定的 accessibility refs
  • 支持多实例：可同时运行多个浏览器实例

2. 配置

安装 PinchTab

# macOS / Linux
curl -fsSL https://pinchtab.com/install.sh | bash

# 或 npm
npm install -g pinchtab

启动服务

pinchtab &

服务启动后访问 http://localhost:9867

3. 使用

基础命令

# 打开网页
pinchtab nav https://example.com

# 获取页面结构
pinchtab snap -i -c

# 点击元素
pinchtab click e5

# 输入文字
pinchtab fill e3 "搜索内容"

# 截图
pinchtab screenshot

4. 推荐理由

  1. 省 Token：800 tokens/page
  2. 操作稳定：accessibility refs
  3. 轻量快速：12MB
  4. 本地安全
